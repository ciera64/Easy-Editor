Aplikasi ini adalah proyek Photo Editor berbasis desktop yang dikembangkan menggunakan Python dan framework PyQt5 di VS Code untuk keperluan pengolahan gambar secara digital. Aplikasi ini mengintegrasikan antarmuka grafis (GUI) yang interaktif untuk memudahkan pengguna dalam melakukan editing foto, seperti penyesuaian filter, rotasi, hingga membuatkannya hitam-putih dengan memanfaatkan efisiensi pemrosesan dari library Python.
